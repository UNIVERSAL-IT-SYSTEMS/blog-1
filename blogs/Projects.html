WorksOfBarry has projects in the works and we're always looking for developers to help contribute!

  • Relic Package Manager is a package manager for the ILE environment. It's mainly a build tool, but includes the functionality to download a ZIP file from the web and then run the build file. We currently have 25 or so projects, but sadly they're scattered. This project is used within the OSSILE repository, maintained by IBM.
  • Giti is a Git repository browser for 5250. Currently it allows you to browse repos, view commits (+ revert) and branches (and checkout). More features are coming like creating, viewing and reseting to tags, as well as a git status screen.
  • soon to come open-source code analyser for free-format RPG. Main feature is going to be that it cleans up sub-routines, procedures and data-structures and variables that are not references - as well as places you could possibly refactor code (if,elseif,elseif,endif turned into select,when for example).
