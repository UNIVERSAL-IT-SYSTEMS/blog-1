So I have migrated my blog to GitHub pages. Tumblr had ruined all my code examples and code examples are the key points to my blog entries.

It seemed like Tumblr did some kind of processing of the custom HTML before it returns it to the client. I spend about two weeks trying to fix it with no luck so I left for GitHub.

Now I get full control over the content and the way it looks, just how I like it! Welcome to the new WorksOfBarry.
