Project Alexis is a college project for the last unit on my course - Unit 311, Create a Procedural Program. Creating procedural programs is easy and boring! So I took it to the next step and made a stack based virtual machine in RPGLE on IBMi.

So far, it's a very basic instruction set. It's probably best to explain by showing examples and listing its features.
Features include:

  • Functions
  • Implicit types
  • Interpreted
  • Debugging tools

First off, functions. Everything is inside a function, there are no global variables or anything like that. The two function instructions are STRFUN and ENDFUN - as you can guess, start function and end function.

Next is implicit types. You don't have to worry about type conversion as it will be converted at runtime - everything is stored as Char(72) in the backend. The only thing you need to note is that each type has it's own operations. For example, for adding doubles (decimal numbers) you'd do a DECADD or adding two integers you'd do an INTADD.

It's interpreted, and there's nothing wrong with that.

The best part, it has a debugger. It allows you to step through, set breakpoints.. the lot.

Better start with some examples, with comments: 

STRFUN Test
  .LDVAL Hello world     //Load 'Hello world' onto the stack
  DUP                    //Duplicate the latest value
  PRINTF                 //Print the lastest value on the stack

  .LDVAL *BLANK          //Load '' onto the stack
  CALL  HELLO            //Call a function named HELLO (below), all functions require a parameter.

  .LDVAL 1               //Load '1' onto the stack
  .LDVAL 5               //Load '5' onto the stack
  SUBSTR                 //Do a subst
  PRINTF                 //Print the latest value on the stack
ENDFUN

STRFUN HELLO
  .LDVAL Hello there
  DUP
  PRINTF

  .LDVAL 1
  .LDVAL 5
  SUBSTR
  .LDVAL 7
  .LDVAL Liam
  APPEND
  PRINTF
ENDFUN
